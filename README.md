# PG Manager

## Getting started

```bash
git clone https://github.com/Binary-Store/pg_manager_server.git
cd pg_manager_server
npm install
npx dbmate up
npm run docs
npm start

# to run in Development
npm run dev # will provide hot reload
```

### API documentation

to see API docs you need to first generate build by running command `npm run docs` will be available on route below
