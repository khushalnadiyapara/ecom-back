# Delete Guest Profile Image

### URL : /api/pg/`:pgId`/guest/`:guestId`/profile-image

### Method : `DELETE`

---

### Description

Delete the guest profile image by id.

---

### Request

```json

"header": {
  "Authorization" : "String" // required (Token)
}

```

---

### Response

### status : `204` : Profile Image is deleted successfully but no content to return.
