# Guest Document Delete

### URL : `/api/pg/`:pgId`/guest/document/`:guestId`

### Method : `DELETE`

---

### Description

will delete document of Guest.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}

---

### Response

status : `204`

- not content return.

---
```
