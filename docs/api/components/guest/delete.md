# Delete Guest

### URL : /api/pg/`:pgId`/guest/`:guestId`

### Method : `DELETE`

---

### Description

will delete the guest with the given id.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}
```

---

### Response

status : `204`

- not content return.

---
