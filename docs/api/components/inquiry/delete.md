# Delete Inquiry

### URL : /api/pg/`:pgId`/inquiry:`inquiryId`

### Method : `DELETE`

---

### Description

will delete the inquiry of given id.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}
```

---

### Response

status : `204`

- Operation successful, but no content return.
