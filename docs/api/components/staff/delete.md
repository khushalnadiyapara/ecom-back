# Delete staff

### URL : /api/pg/`:pgId`/staff/ `:staffId`

### Method : `DELETE`

---

### Description

will delete the staff with the given id.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}
```

---

### Response

status : `204`

- not content return.

---
