# Monthly Payment Delete

### URL : /api/pg/`:pgId`/monthly-payment/`:monthlyPaymentId`

### Method : `PUT`

---

### Description

will delete the monthly payment with the given id.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}
```

---

### Response

### status : `204`

- Operation successful, but no content return.
