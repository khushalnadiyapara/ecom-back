# Delete booking

### URL : /api/pg/`:pgId`/booking/`:bookingId`

### Method : `DELETE`

---

### Description

will delete the booking with the given id.

---

### Request

```json
"header": {
  "Authorization" : "String" // required (Token)
}
```

---

### Response

status : `204`

- not content return.

---
